# applestore_watcher

## uasge 
- edit `target.json`
-  `npm install -g`
- `applestore`

## rules
- target.json [serial:name]
  
  example: 
  
  ```
  {
    "MMEF2CH/A": "AirPods"
  }
  "MMEF2CH/A": serial id of airpods (http://www.apple.com/cn/shop/product/MMEF2CH/A/airpods)
  "AirPods": display name
```
